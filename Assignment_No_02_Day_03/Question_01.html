Explore and explain the various methods in console function

Console function methods:
1. console.assert() : The console.assert() method writes a message to the console, but only if an expression evaluates to false.
2. console.clear() : Clear all messages in the console.
3. console.count() : It counts the console messages.
4.The console.error() method writes an error message to the console.
5.The console.group() method indicates the start of a message group.
6.The console.groupCollapsed() method indicates the start of a collapsed message group.
7.The console.info(), console.log() method writes a message to the console.
8.The console.table() method writes a table in the console view.
9.The console.time() method starts a timer in the console view.
10.The console.trace() method displays a trace that show how the code ended up at a certain point.
11.The console.warn() method writes a warning to the console.
