Difference between var lets const:
Keyword Function vs Block code Redefinable
Var : JavaScript variables are containers for storing data values. Function Code YES
Let : Variables declared with the let keyword can have Block Scope. Block Code YES
Const Block Code No

Example for var:
var x = 5; var y = 6; var z = x + y;
Example for Let:
var x = 10; // Here x is 10
{
let a = 2;
// Here x is 2
}
// Here x is 10
Example for Const:
var b = 10; // Here b is 10
{
const f = 2;
// Here f is 2
}
// Here b is 10
